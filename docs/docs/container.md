# Container

A container is a responsive block element used with breakpoints.

## Example

```html
<div class="container">
  <p>Text Content</p>
</div>
```


## Full Container

if you want a container which with `max-width: 100%`, easily give if a `full` class

```html
<div class="container full">
  <p>Text Content</p>
</div>
```
