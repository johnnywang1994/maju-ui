# Sizing

## width, height

- responsive: false

### format
`w-{percent}`, `h-{percent}`

- allowed {percent} value:  
`25, 50, 75, 100`

### Example
```html
<div class="container">
  <div class="box w-100"></div>
</div>
```


## max-width, max-height

- responsive: false

### format
`mw-100`, `mh-100`

### Example
```html
<div class="photofix">
  <img class="d-block mw-100" />
</div>
```
