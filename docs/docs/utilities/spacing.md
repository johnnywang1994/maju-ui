# Spacing

- responsive: true

### format
`m-{level}`, `p-{level}`

- {level}: `1-5`, each level stands for `4px`.

### Example
```html
<div class="box m-1">
  Content
</div>
```

### format
`m-{breakpoint}-{level}`, `p-{breakpoint}-{level}`

### Example
```html
<div class="box m-1 m-md-2 m-lg-4">
  Content
</div>
```
