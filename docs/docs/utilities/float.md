# Float

- responsive: true

### format
`float-{align}`

- allowed {align} values:  
`left, right, none`

### Example
```html
<div class="float-right">
  Content to Float
</div>
```

### format
`float-{breakpoint}-{align}`

### Example
```html
<div class="float-right float-sm-left">
  Content to Float
</div>
```
